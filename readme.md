#StarboundCannery
##Cannery mod for Starbound
Adds the ability to can certain foods. Food with effects still has effects. Cans stack to 10. Food effects persist. Uses .patch files to add canning recipes to the appropriate precursor foods.
###Common
* Fish
* Corn
* Mushrooms
* Pearlpeas
* Tomato
* Pineapple

###Uncommon
* Fish Stew
* Meat Stew
* Shepherd's Pie
* Pearlpea Risotto
* Mushroom Rice

###Rare
* Oculemon Stew
* Fruit Salad
* Automato Stew

###Ledgendary
* Wartweed Stew
* Pearlpea Parcels
* Ultimate Juice
